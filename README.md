## Requirements
- Aircrack-ng 
- Network Adapter which supports monitor mode and packet injection
- Python

## Using The Tool 

Setting up the files 
```bash
git clone https://github.com/Akshayan-N/Wifi-Cracker.git
cd Wifi-Cracker
pip install -r requirements.txt
```

Usage
```bash 
sudo python wificracker.py
```
